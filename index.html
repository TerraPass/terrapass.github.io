Description:

This non-interactive demo shows bots, controlled by a very basic AI, playing a simplified version of basketball.

AI is based on state machines with messaging and uses steering behaviours for game agents' movement, described in M.Buckland's "Programming Game AI by Example".

Robot model, that represent players, was borrowed from Unity's Mecanim Tutorial. All other 3d assets I created myself using Blender.

Players are animated using Unity Mecanim system.

A basic implementation of a table tennis game.

Click and hold left mouse button to aim your service, after that simply move your paddle around with a mouse and attempt to score.
